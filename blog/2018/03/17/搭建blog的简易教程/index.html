使用hexo(建立博客页面及内容)+ next(适合写博客的主题)+ github.io(在其中部署) 即可完成简单博客的建立。

hexo安装

安装Node.Js

我们需要nodejs来生成静态页面,因此先在nodejs官网下载安装

安装git

其次，还需要通过git，把本地的hexo内容提交到github上去

  • git下载，安装过程所有选项默认项就行
  • git bash的使用： 在任意目录下，点击右键，有git bash选项，接下来的命令都是在git bash中输入的

hexo安装

  • 在任意目录打开git bash，执行命令npm install -g hexo
  • 选择一个位置创建Hexo文件夹，如“D:\project\hexo”
  • 在该目录打开git bash，执行命令hexo init，Hexo 即会自动在目标文件夹建立网站所需要的所有文件

使用命令

在hexo的目录下，打开git bash,在其中输入命令

  • 清除之前生成的静态页面
    hexo clean
  • 生成静态页面
    hexo g
  • 发布到git.io
    hexo d
  • 其它操作
    在发布之前，可以使用hexo s命令启动本地部署（输入ctrl+c停止）
